# Hotel Management System (Python)

A beginner-friendly Python CLI project that demonstrates
real-world concepts like authentication, booking, and payments.

## Features
- User registration and login
- Room booking
- Check-in / Check-out
- Payments

## Tech Stack
- Python 3
- JSON
- File Handling

## Project Structure
hotel_project/
├── auth_json.py
├── general.py
├── greet.py
├── users.json
└── README.md

## How to Run
python general.py
